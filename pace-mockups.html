Pace visualization — three mockups

Each option replaces the per-family "Days Between Releases" chart with a single unified pace visual.

Option 1

Sawtooth — "days since last release"

The y-axis is days since any Claude release. It resets to zero on each ship, then climbs one unit per day. Every valley is a release (dot colored by family). Lower peaks over time mean Anthropic is shipping faster.

Option 3

Cumulative staircase

Total Claude releases shipped over time. Every step up is a ship; the steeper the slope, the faster the pace. Three stacked bands show the contribution of each family. Dashed reference lines mark "1 release / month" and "1 / quarter" for orientation.

Opus Sonnet Haiku
Option 4

Beeswarm timeline

Each release is a dot on a single time axis. Dots nudge upward when releases cluster in time — so the densest columns are the busiest weeks. Colored by family.

Option A · days between releases

Gap scatter with trend

Each dot is a release, positioned at its date (x) and the days since the previous ship from any family (y). The fitted trend line shows the gap shrinking over time. If Anthropic keeps accelerating, the line slopes down and to the right — which it does.

Option B · by quarter

Releases per quarter

One bar per calendar quarter, height = number of ships, stacked by family. The 2024 bars are lonely; 2025 starts filling in; 2026 is the loudest year yet — and Q2 isn't done.

Staircase with quarter bands
