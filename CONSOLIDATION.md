# Site Consolidation Guide

This branch consolidates the **daily-briefing-podcast** and **jacobs-research-journal** repos into the general site.

## New Structure

```
general/
├── styles.css                    (existing base styles)
├── styles-consolidated.css       (NEW: Inter+Lora fonts, research/briefing components)
├── about.html                    (updated nav: 8 tabs)
├── now.html                      (updated nav + Projects section)
├── reads.html                    (updated nav)
├── .gitattributes                (Git LFS for audio files)
├── .nojekyll                     (GitHub Pages subdirectory support)
├── scripts/
│   └── transform_article.py      (article migration script)
├── briefing/                     (Briefing - podcast)
│   ├── index.html                (landing page with episode listing)
│   ├── feed.xml                  (podcast RSS - renamed "Briefing")
│   └── posts/
│       └── episode-*.html        (12 per-episode pages)
├── research/                     (Research - articles)
│   ├── index.html                (landing page with article cards)
│   └── feed.xml                  (research RSS feed)
```

## Remaining Steps (Local Git Required)

### 1. Migrate Research Articles
```bash
# From the general repo root:
mkdir -p research/raw
cp ../jacobs-research-journal/posts/*.html research/raw/
python3 scripts/transform_article.py research/raw research/posts
git add research/posts/
```

### 2. Copy Research Images
```bash
cp -r ../jacobs-research-journal/images research/images
git add research/images/
```

### 3. Copy Podcast Assets
```bash
# Artwork
cp -r ../daily-briefing-podcast/artwork briefing/artwork
cp ../daily-briefing-podcast/podcast-cover.jpg briefing/artwork/

# Transcripts
cp -r ../daily-briefing-podcast/transcripts briefing/transcripts

# Audio (tracked by Git LFS)
git lfs install
cp -r ../daily-briefing-podcast/episodes briefing/episodes
git add briefing/
```
